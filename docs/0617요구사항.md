## 버그 (Bug)

| #    | 내용                                                                 | 원인                                                                             | 상태                            |
| ---- | -------------------------------------------------------------------- | -------------------------------------------------------------------------------- | ------------------------------- |
| B-01 | 폴더 드래그앤드롭 시 폴더 중복 생성                                  | `dstPath` 대신 `dstNode.path` 사용해야 함                                        | ✅ 수정 완료                    |
| B-02 | "외부에서 파일이 변경되었습니다" (409) 자동저장 중 반복 발생         | `handleBodyChange`가 `push` + 직접 `triggerSave` 이중 호출 → 동일 SHA로 동시 PUT | ✅ 수정 완료                    |
| B-03 | 레포 전환 후 같은 경로 노트 저장 시 409 발생                         | `noteCache`가 이전 레포의 SHA를 유지하여 새 레포에서 충돌                        | ✅ 수정 완료 (계속 테스트 필요) |
| B-04 | 레포 전환 후 이전 레포의 폴더 트리가 사이드바에 잔류                 | `fileTreeStore.tree`를 레포 전환 시 초기화하지 않음                              | ✅ 수정 완료                    |
| B-05 | 기존 노트(frontmatter 없는 마크다운)를 열면 제목이 "Untitled"로 표시 | `parseFrontmatter`가 frontmatter `title` 없을 때 H1·파일명 폴백 없음             | ✅ 수정 완료                    |
